# 问答
### 能否保存敏感数据？
我至少自己就在保存，但我也不敢保证绝对安全。

现在自己组建虚拟局域网来保证数据处于内部，保证数据安全，但几年的外网使用也没出安全问题。

可以运行frp服务，可以网上找个免费的。加群可以了解这方面的信息。
